# Playbook: Beginner Handoff

## Goal
Return work in a form a non-developer can actually use.

## Rules
- prefer plain English over jargon
- give one recommended path
- use ordered steps
- include copy-paste commands where useful
- say what the user should expect to see
- say what to do if it fails

## Handoff template
- What I changed
- Why I changed it
- How to run it
- What success looks like
- What to do if you get stuck
- Best next upgrade

## Tone
Calm, practical, and non-patronising.
